# Workflow

1. confirm acceptance criteria
2. confirm tests and checks
3. review CI/CD impact
4. review migration safety
5. review rollback path
6. review observability impact
7. review known risks

## Output

- go/no-go recommendation
- blockers
- mitigations
- rollout notes
- rollback notes
