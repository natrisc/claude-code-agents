# Workflow

1. identify trust boundaries
2. review authentication and authorization impact
3. review input validation
4. review secret handling
5. review data exposure risks
6. review dependency and supply-chain concerns
7. review logging and audit implications

## Output

- findings by severity
- required fixes
- optional hardening
- release blocker: yes/no
